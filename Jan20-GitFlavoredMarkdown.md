#CSS Code Coloring in GFM

```
```CSS
.highlight {
  color: ##900;
  background-color: #ffc;
}

#facebox {
  margin: 0 0 0 0;
  padding: 6px 3px;
}

span.red {
  color: red;
}
\```

\```HTML
<div id="facebox" class="highlight">
  <span class="red">
    Hello World!
  </span>
</div>
\```
```

*Ignore the backslashes above. They are meant to force the ``` to display rather than render.*
